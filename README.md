# Black Pearl

A simple JavaScript project.

## How to use

1. Clone this repository.
2. Open the `index.html` file in a web browser.
3. Press `F12` or `Ctrl + Shift + I` (or `Cmd + Option + I` on Mac) to open the browser's Development Console.
4. Enjoy!

## Requirements

- Web browser
